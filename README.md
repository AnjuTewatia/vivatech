# Task Manager

Task Manager is a simple React application for managing tasks. It allows users to view existing tasks, create new tasks, and delete tasks.

## Table of Contents

- [Features]
- [Gettting start]

## Features

- View a list of existing tasks
- Create new tasks
- Delete tasks
- Export the list to the excel file
